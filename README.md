Self Growth
===========

## Description

An Application to keep track of personal growth progress .

## Major features

* **Personal OKR**: Personal goal and progress tracker 
* **Diary**: Keeps track of user's personal growth progress

## Style guides

### Git

#### Branching

- Follow Git flow branching

#### Commit messages

- Use English for all commits
- Use the present tense ("Add feature" not "Added feature")
- Limit the first line to 72 characters or less
- Commit message's first line starts with an uppercase character (e.g "Initial commit" not "initial commit")

### Java Spring convention

- Java convention.
